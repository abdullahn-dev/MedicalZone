# 🏥 MedicalZone - AI-Powered Healthcare Ecosystem

A comprehensive Android application built with **Java/Firebase** and integrated with **Google Gemini 1.5 Pro** for proactive disease detection.

## 🌟 Key Modules
* **👨‍⚕️ Doctor Dashboard:** Manage patient records, appointments, and diagnostic history.
* **👤 Patient Portal:** Access personal health data, schedule consultations, and run AI health checks.
* **🛡️ Admin Panel:** Complete system oversight, user verification, and database management.

## 🧠 Special Feature: AI Parkinson's Detection
Integrated an AI model to analyze patient symptoms and provide early-stage detection alerts using the Gemini API.

## 🛠️ Tech Stack
* **Frontend:** Android Java (XML Layouts)
* **Backend:** Firebase (Auth, Firestore, Storage)
* **AI Engine:** Google Gemini 1.5 Pro
